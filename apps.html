Installing apps

Since the device can't be activated, you won't be able to install apps from appstore and impactor/computer. Anyway here are some alternative ways to get apps.

Solution #1 - Jailbreak + AppSync + IPA Installer

This is the most valid solution, but as I said in other tutorials, jailbreaking is hard and buggy.

But if you got it to work, you can install AppSync Unified from cydia.angelxwind.net repo and IPA Installer from bigboss repo.

Then copy an ipa on device with any tool and install it with IPA Installer

Solution #2 - Copying as system app

That's a bit tricky and not the best solution, but it will work 90% of the times

Take an ipa, open it with 7zip or winrar, extract payload and get the .app folder

Copy that folder in /Applications of iOS 7 system partition

Set correct permissions for app executable (should be 775, check anyway)

Run uicache or reboot

Solution #3 - Signing services

Won't probably be useful for many people, but I decided to include it here since it works anyway

With any signing service, sign your ipa and put it on a website so that you can install it by ota-distribution
